## Descripción

Esta es una página web tributo al universo de **Batman**, diseñada con una estética oscura, moderna y responsiva.

**La particularidad de este proyecto:** Todo el código (estructura, diseño y lógica) fue generado utilizando Inteligencia Artificial con **tan solo 3 prompts** (instrucciones). El objetivo era probar la eficiencia de la IA generativa para crear prototipos web funcionales en tiempo récord.

## Características

* **Modo Gotham:** Diseño oscuro inmersivo inspirado en la estética de las películas.
* **Diseño Responsivo:** Se adapta perfectamente a móviles y escritorio.
* **Estilos Modernos:** Uso de CSS3, gradientes y tipografías temáticas.
* **Ligero:** Sin dependencias externas pesadas, solo HTML, CSS y JS puro.

## El Desafío: 3 Prompts

Para construir este sitio, se utilizó un modelo de IA (puedes especificar aquí si fue Gemini, ChatGPT, etc.) y una secuencia estricta de 3 pasos:

1.  **Prompt 1 (Estructura): Generación del esqueleto HTML semántico y contenido base sobre la historia de Batman.
2.  **Prompt 2 (Estilo): Creación de una hoja de estilos CSS impactante, con paleta de colores negra/amarilla y efectos visuales.
3.  **Prompt 3 (Detalles/Interacción):** Refinamiento del diseño, corrección de errores y añadido de interactividad básica con JavaScript.
